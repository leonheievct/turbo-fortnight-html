Название const(константы) должно совпадать в нижнем и верхнем регистре с названием фаила все фаилы .jsx

ПРИМЕР:

import React from 'react'

const NavBar = () => {
    return (
        <div>

У нас есть базовая кострукция, все теги что мы будем писать должна находиться внутри родного блока div его нельзя трогать



----------------------

Базовые теги
h1-h6 - Теги заголовка(Обозначает оглавление блоков,название саитов и т.д)
p - Пораграф (обычный текст)

a - Гипер ссылка(Обычная ссылка)
-Полная перезагрузка страницы React теряет все состояния и падает производительность

Link - Гипер ссылка()
-Переход без перезагрузки старницы оставляет состояния React подгружает только измененные части

<ul> - Открытия списка
    <li> - Перечесление эти тегом

    </li>
</ul> - Зактрытие списка

---------------------------

-Картинки img - (в src внутри тега указываеться путь до Картинки)
-Для продвинутых картинок с адптивами тег picture(внутри тег source)

Главный часто используемый тег без которого не чего не собираеться div
div - является невидимои коробкой в которую можно положить любые теги и любые коробки
все теги можно вкладывать друг в друга сколько угодно раз главное соблюдать табуляцию
