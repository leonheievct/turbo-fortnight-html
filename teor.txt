Создаение клиентской части 
- Для создания проекта требуеться ввести в терминале npm create vite@latest
- Нажимаем 'у' для подтверждения создания проекта 
- Полсе вводим папки клиентсой части Client
- после выбираем фреемворт разработы React
- После выбираем язык разработки JavaScript(в дальнейшем будет изменен на надстроику TypeScript)
- При предложение эксперементал (no)

-------------------------

В основной папке src находяться главные фаилы
App.jsx - Корневой компанент (оно собирает все компоненты)
main.jsx точка входа в наше приложения (собирает наш App в нужном моде)

-------------------------

Для создания быстрого компанета исп команда
rafce - создание стрелочной функций (Требует ES + снипет (16.5 миллионов скачек))
